学习使用 unset

    unset
    1. 如果这个属性本来有从父级继承的值,则将该属性重新设置为继承的值
    2. 如果没有继承父级样式,则将该属性重新设置为初始值
    典型的使用场景:
    重置 box-sizing: border-box => content-box
    轮播图按钮(见下)
// 设置轮播图按钮 .box::before, .box::after { content: "<"; position: absolute; top: 130px; left: 20px; line-height: 40px; width: 40px; height: 40px; border-radius: 50%; border: 1px solid #000; text-align: center; cursor: pointer; } // 为了调整右箭头, 之前的代码(因为left被定义了 移动到右边就要重新计算 很麻烦) .box::after { content: ">"; left: calc(100% - 60px); } // 如果我们使用 unset 一切变得很清晰明了 .box::after { content: ">"; /* 这里的 unset 就是 left: auto; 所以 auto 也可以 */ left: unset; right: 20px; }
