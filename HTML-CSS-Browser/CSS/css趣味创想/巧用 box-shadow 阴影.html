巧用 box-shadow阴影

可以说 box-shadow 不仅仅可以用来产生阴影 这个属性也是最方便快捷不依赖多元素的复制原有图形轮廓的办法 而且还要了解到: - 阴影可以是多重的 - 阴影的动画

// DEMO1 .new-moon { width: 150px; height: 150px; border-radius: 50%; // 无需设置 元素本身的颜色 使用 inset 内阴影画出月亮 box-shadow: -20px -10px 5px yellow inset; }
// DEMO2 用阴影绘制云朵 .cloud { width: 100px; height: 60px; margin: auto; background: #fff; border-radius: 50%; // 阴影是可以多重的 box-shadow: -50px 5px 0 #fafafa, 60px 10px 0 #fafafa, 45px -10px 0 #fafafa, 0px -30px 0 #f1f1f1; }
// 单元素纯 CSS 实现以下效果 .circle { width: 150px; height: 150px; border-radius: 50%; background-color: purple; box-shadow: 100px 0 0 #f90, 100px 0 0 orangered inset; }
// 使用 filter: drop-shadow() 整块阴影 .arrow { position: absolute; top: 20%; left: 20%; border: 30px solid #F90; border-left: 30px solid transparent; border-top: 30px solid transparent; transition: filter 0.3s; cursor: pointer; } .arrow:hover { // 无法使用 box-shadow 透明边框也会出现阴影 /* box-shadow: 0 0 6px #f90; */ // drop-shadow 会识别图形实际可见的部分实现加阴影 filter: drop-shadow(2px 5px 8px #f90); } .arrow::after { content: ''; position: absolute; top: -17px; left: -94px; width: 118px; height: 38px; border-radius: 50%; box-shadow: -29px 5px 0 #f90 inset; }
// drop-shadow 模拟不规则边框 .pointer { width: 150px; height: 50px; background-color: brown; filter: // 四次叠加 drop-shadow(0 0 .5px #fff) drop-shadow(0 0 .5px #fff) drop-shadow(0 0 .5px #fff) drop-shadow(0 0 .5px #fff); } .pointer::after { content: ''; position: absolute; top: 0; right: -72px; border: 50px solid brown; border-right: 22px solid transparent; border-top: 25px solid transparent; border-bottom: 25px solid transparent; }
