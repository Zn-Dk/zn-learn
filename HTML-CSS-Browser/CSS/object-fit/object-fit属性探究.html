object-fit.（容器宽高分别为200px、400px,带边框）选择了一张大于容器宽度，一张小于容器宽度的图，图片宽高都设置为100%

object-fit 的作用对象: img

无 object-fit ;
内容可能会被拉伸以完全适应容器的宽高。和不使用该属性的效果一致。

object-fit: fill ;
默认，不保证图片保持原有的比例，内容拉伸填充整个内容容器。

object-fit: contain;
保持图片原有尺寸比例。内容被缩放，覆盖最短的一边。

object-fit: cover
保持图片原有尺寸比例。内容被缩放，覆盖最长的一边。

object-fit: scale-down
保持原有尺寸比例。内容的尺寸与 none 或 contain 中的一个相同，取决于它们两个之间谁得到的对象尺寸会更小一些。

object-fit: none
保留原有元素内容的长度和宽度，也就是说内容不会被缩放。
