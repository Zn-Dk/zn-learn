举例说明为什么 transform 比改变 true position 好

- transform 的影响发生在 渲染-GPU进程的合成线程上, 即只涉及光栅化的计算 - position 改变是直接触发 reflow , 进而导致全部样式重新计算, - 当js代码发生阻塞时, 动画效果随着停止 PS. 滚动条也是属于合成线程, 所以freeze时依然可以滚动页面
