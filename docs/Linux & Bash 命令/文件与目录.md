In Linux everything is a File
> Linux 系统一切皆是文件

Linux is an Extensionless System
> Linux 系统是无扩展名的系统,(不看是否是 .exe)

Linux is a Case Sensitive System
> Linux 系统是大小写敏感的

> 所有命令后面空格 然后双击 两个 Tab 可以自动补全
> 比如 ls 将列出 ls 开头的所有命令、文件等
> ls -- 双击 Tab 自动列出 --可用的参数

## touch
touch + 文件名, 创建文件

## cat
cat + 文件名, 列出文件内容

## file
file + (一个或者多个文件/目录名, 空格分隔), 查看文件类型
> 文件存在空格怎么正确引用?
> 1. 用引号括起来, 比如 file "file name"
> 2. 用反斜杠转义, 比如 file file\ name

## ls
ls + 文件/目录名, 查看文件/目录信息

## cd
cd + 文件/目录名, 切换到指定目录

## pwd (print working directory)
pwd, 查看当前目录

## mkdir
mkdir + 文件/目录名, 创建文件/目录

## rmdir
rmdir + 文件/目录名, 删除空文件/目录

## rm
rm + 文件/目录名, 删除文件/目录

## / 目录文件夹简要介绍

- bin (binary)
- boot (boot file)
- sbin (super user binary)
- lib (library)
- dev (device | 设备配置文件)
- etc (etcetera | 一般的配置文件)
- tmp (temporary)
- var (variable)
- proc (process)
- sys (system)
- home (home directory)
- root (root directory)
- boot (boot directory)
- media (media directory | 挂载的设备(比如U盘))
- mnt (mount directory)
- opt (optional directory)
- selinux (security enhanced linux)
- srv (service directory)
- tmp (temporary directory)
- usr (unix software resource | unix 操作系统软件资源, 不要认为是 user 的简称)
- var (variable directory | 缓存, 软件运行产生的文件等)
